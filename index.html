QoS Risk Skoru

Enerji aksiyonlarının güvenliğini değerlendirmek için beş boyutlu ağırlıklı kompozit skor. Pilot verisiyle kalibre edilen ağırlıklar, kriz modunda Drop ve HO bileşenlerini otomatik artırır.

// risk_score formülasyonu
Risk_Score =
  w₁·Drop_pred_norm
 + w₂·(1 − HO_success_pred_norm)
 + w₃·PRB_util_pred_norm
 + w₄·(1 − SINR_pred_norm)
 + w₅·Backhaul_util_norm
Karar Kuralı:
Risk_Score > τ → Enerji aksiyonları durdurulur / rollback
Risk_Score ≤ τ → Kademeli enerji aksiyonları uygulanır
Parametre Ağırlıkları (Normal Mod)
Drop Rate
w₁=0.4
HO Başarı
w₂=0.3
PRB Kullanım
w₃=0.1
SINR
w₄=0.1
Backhaul
w₅=0.1
⚠ Kriz Modunda Ağırlık Artışı
Drop Rate → w₁ = 0.5
HO Başarı  → w₂ = 0.4
Sistem daha korumacı karar alır
